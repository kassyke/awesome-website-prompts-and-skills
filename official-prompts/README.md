# Prompts（官方示例 Prompt）

> Tencent EdgeOne 维护的**建站提示词合集**——复制一段 Prompt 喂给 AI，即可生成并部署一个完整网站。

## 如何使用

1. 挑一个你喜欢的 Prompt，复制全文
2. 粘贴到 **[WorkBuddy](https://www.codebuddy.cn/work/)** / Claude Code / Cursor 等 AI 编程工具
3. AI 会自动生成完整项目（含代码、样式、Edge Functions）
4. 按 AI 的指引部署到 [EdgeOne Pages](https://pages.edgeone.ai/)，通常 3 分钟内可访问
