## 📖 这个仓库是什么

<br/>

这个仓库是 **Tencent EdgeOne** 维护的 **网站资源合集**。

<br/>

在 AI 编程工具中，**给它一个 Prompt 或一个 Skill**——

<br/>

就能生成一个**精美酷炫的网页**，或者带有**登录、支付、AI 能力**的完整网站。

<br/>

仓库中包含两类资产：

<br/>

| 类型 | 是什么 | 怎么用 |
|:---|:---|:---|
| 🧠 &nbsp; **Prompts** | 直接喂给 AI 编程工具的**建站提示词** | 复制 Prompt → 粘贴给 AI → AI 帮你生成并部署完整网站 |

## 🚀 快速开始：三步拿到一个可运行的网站

### 第 1 步：复制下面这段「一键口令」

直接把下面这段话复制，粘贴给你正在用的 AI 编程工具（[WorkBuddy](https://www.codebuddy.cn/work/) / Claude Code / Cursor 等）：

````
请访问这个 GitHub 仓库：
https://github.com/TencentEdgeOne/awesome-website-prompts-and-skills

阅读里面所有的 Prompt 和 Skill 示例，理解它们的结构和玩法，然后：

1. 推荐 2-3 个最适合我的示例
2. 我选定后，基于该示例帮我生成一个完整项目
3. 部署到 EdgeOne Pages（https://pages.edgeone.ai/）
4. 把访问链接给我

我想做一个：___（在这里描述你的网站，例如：AI 图片生成工具 / 旅拍摄影师作品集 / 独立游戏官网 / 在线卖课 SaaS）
````

### 第 2 步：AI 会自动生成代码

AI 阅读示例后，会按你的需求生成完整项目（含代码、样式、Edge Functions）。

### 第 3 步：部署到 EdgeOne Pages

按 AI 的指引一键部署到 [EdgeOne Pages](https://pages.edgeone.ai/)，一般 **3 分钟内** 可访问。

<br/>

> 💡 推荐使用 **[WorkBuddy](https://www.codebuddy.cn/work/)**——国内开发者可直接使用，原生支持 Skills，和 EdgeOne Pages 无缝协作。

## 🌩️ 关于 EdgeOne Pages

[**EdgeOne Pages**](https://pages.edgeone.ai/) 是腾讯面向全球开发者的 **Serverless 全栈部署平台**，服务于 AI 应用、Web 应用、静态站点的一键部署：

- ⚡ **一键部署**：Next.js / Vite / React / Vue / Astro 等主流框架开箱即用
- 🧠 **Edge Functions**：离用户最近的边缘节点跑后端逻辑，低延迟响应
- 🗃️ **KV Storage**：边缘侧轻量存储，适合会话、缓存、用户状态
- 🤖 **AI 友好**：原生适配 AI SaaS / AI Agent / AI Chatbot 场景
- 🌍 **全球加速 & 安全**：基于腾讯全球 CDN + WAF
