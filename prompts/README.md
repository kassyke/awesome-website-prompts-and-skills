# Prompts

> 可以直接喂给 AI 编程工具的「一段 Prompt，一个完整网站」建站提示词合集。

## 如何使用

1. 挑一个你喜欢的 Prompt，复制全文
2. 粘贴到 **[WorkBuddy](https://www.codebuddy.cn/)** / Claude Code / Cursor 等 AI 编程工具的对话框
3. AI 会自动生成完整项目（含代码、样式、Edge Functions）
4. 按 AI 的指引部署到 [EdgeOne Pages](https://pages.edgeone.ai/)，一般 3 分钟内可访问
